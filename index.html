Teverun Fighter Mini (eKFV) Unlock

disconnected

Live lock/unlock over Web Bluetooth. Works in Bluefy (iOS) or Chrome (Android/desktop).

How to start

  1. Download the firmware below and flash it once with the official Teverun app (one time per scooter).
  2. Tap Connect and pick your scooter.
  3. Use Unlock / Lock and set wheel diameter + cruise.

New here? See how to use it in the guide.

Controls

The button shows the action for the current state: Unlock when the scooter is locked (FIN starts with TDE), Lock when it is open. Unlock re-applies your saved wheel + cruise; Lock forces wheel to 10 and turns cruise off.

Settings

Wheel + cruise are saved on this device and re-applied automatically after Unlock.

Firmware

Download firmware

Flash it once with the official Teverun app. It enables the live-toggle this page controls.

Hardware: This firmware is built for the IVCU hardware version 5.x only. It may also run on the "Ali box", but that is untested and not supported - flashing it there is entirely at your own risk.

Identity (FIN)

Editable only after a FIN was read from the scooter. Fallback only - use the Unlock / Lock buttons. A wrong identity breaks the live-toggle (recoverable: type the correct FIN back).

Unlocked with the Android app? This page cannot re-lock it - lock it once in the Android app, then this page works again.

Home-screen shortcut (iOS / Android)

Open the page already set to lock or unlock - a paired scooter reconnects without the picker and the action runs by itself. Make one shortcut for Unlock and one for Lock.

Do one clean run without the shortcut first: connect, then use Unlock / Lock and set wheel diameter + cruise once. Only a prior manual run lets the browser store them for the shortcut's unlock to restore.

iOS (Bluefy) - Shortcuts app -> new shortcut -> action Open URLs -> paste a link below -> add it to the home screen (or a Siri phrase). A plain https link opens Safari, which has no Bluetooth; the bluefy:// scheme opens Bluefy.
